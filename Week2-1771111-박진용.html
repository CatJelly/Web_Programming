Week2 Summary

  • HTML(HyperText Markup Language)
    • 웹 페이지 기술을 위한 마크업 언어
    • 웹 페이지 내용, 구조 담당
    • html 태그를 통해 정보 구조화

  • HTML 기본 문법
    • 요소(element) = start tag + content + end tag
    • 속성(attribute) = 요소의 성질, 특징 정의, [이름=값]
    • 주석 = <!--주석-->

  • Semantic Web - 웹 페이지에 메타 데이터를 부여해 의미와 관련성을 가지는 DB로 구축함
    • ex) form, table, img, header, nav, aside, section, article, footer

  • Basic Tags - html, head, body, meta, title, link, script
    • meta example
      • <meta name="keywords" content="HTML, CSS, XML">
      • <meta name="description" content="Web tutorial">
      • <meta http-equiv="refresh" content="30">

    • Text 관련 tags
      • h1, h2, ... h6
      • b, strong, i, em, small, ...
      • mark, del, ins, sub, sup, p, pre, q, blockquote, ...
      • br, hr,  , ...

    • Hyperlink
      • <a href="path">
      • target attribute - _self, _blank, _parent, _top

    • List
      • <ul>, <ol>, <li>
      • type - 1, A, a, I, i, ...
      • type - 1, A, a, I, i, ...

    • Table
      • <table>
      • border, rowspan, colspan
      • tr > td, thead > tr > th, tfoot ...

    • Image, Audio, Video
      • <img>
        • src, alt, width, height
      • <audio>, <video>
        • src, preload, autoplay, loop, controls
        • poster, width, height
